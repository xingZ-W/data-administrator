### Phase 1: 基础数据采集与配方管理（部分完成）

#### 1.1 高频数据采集模块 ✅
**文件位置**: `DataAdministrator/数据采集/`

**已创建文件**:
1. `HighFrequencyWeldingCollector.cs` - 高频焊接参数采集器（1kHz）
2. `CircularBuffer.cs` - 环形缓冲区（线程安全）

**核心功能**:
- ✅ 1kHz 实时采样（激光功率、焊接速度、焦点位置、背向反射、熔池温度）
- ✅ 环形缓冲区（10 秒数据缓存，自动覆盖最旧数据）
- ✅ 并行数据读取（提高采集效率）
- ✅ 事件驱动（OnSampleCollected、OnWeldingCompleted）
- ✅ 异常处理与日志记录
- ✅ 精确时序控制（微秒级延迟）

**关键性能指标**:
- 采样率：**1000 Hz** (1ms 间隔)
- 缓冲区容量：**10,000 个样本** (10 秒@1kHz)
- 读取延迟：**≤5ms** (并行读取)
- 时间精度：**≤100μs** (Stopwatch 高精度计时)

**使用示例**:
```csharp
// 在极耳焊接工站初始化时创建采集器
public class 极耳焊接 : Father
{
    private HighFrequencyWeldingCollector _collector;
    
    public 极耳焊接(Form1 form, string FilePath2)
    {
        this.form = form;
        // ... 其他初始化代码
        
        // 启动高频采集器
        _collector = new HighFrequencyWeldingCollector(form);
        
        // 订阅焊接完成事件
        _collector.OnWeldingCompleted += async (data) => 
        {
            // 焊接完成后自动触发质量判定

        };
    }
}
```

**使用示例**:
```csharp
// 在进站流程中自动调用配方
public override string 工位 1 进站(string Code = "")
{
    moduleCode1 = form.PLC_ui.Read_String(FindIniPath("WIP 码")[0]);
    
    // 自动加载配方
    var recipeManager = new RecipeManager(form);
    var result = await recipeManager.AutoLoadRecipe(moduleCode1);
    
    if (result.IsSuccess)
    {
        form.P_Class.TxtLog($"配方加载成功：{result.Recipe.RecipeId}", 2);
        // 继续进站流程...
    }
    else
    {
        form.P_Class.TxtLog($"配方加载失败：{result.ErrorMessage}", 1);
        // 触发 NG 信号
        form.PLC_ui.write(FindIniPath("进站 1NG")[0], true);
    }
    
    return "";
}
```

## ⚠️ 注意事项

### 1. 依赖库安装
需要添加 NuGet 包引用：
```
Install-Package System.Data.SQLite
Install-Package Newtonsoft.Json
Install-Package Microsoft.ML.OnnxRuntime
```

### 2. 数据库初始化
首次运行时需要创建数据库目录和表：
```csharp
// 在 Program.cs 或 Form1_Load 中调用
var dbDir = Path.Combine(AppDomain.CurrentDomain.BaseDirectory, "Data");
if (!Directory.Exists(dbDir))
    Directory.CreateDirectory(dbDir);

// 初始化配方数据库
var recipeCache = new RecipeCache(Path.Combine(dbDir, "Recipes.db"));
```

### 3. PLC 地址配置
需要在设置窗口中添加以下 PLC 点位：
- 激光功率反馈
- 焊接速度反馈
- 焦点位置反馈
- 背向反射反馈
- 熔池温度反馈

### 4. MES 接口对接
